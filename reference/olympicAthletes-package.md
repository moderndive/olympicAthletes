# olympicAthletes: Olympic Athlete Event Data, Athens 1896 to Milano-Cortina 2026

A tidy, long-format dataset of every athlete-event participation in the
modern Olympic Games. Extends the rgriff23 *Olympic_history* dataset
(1896-2016) with five additional editions scraped from
[olympedia.org](https://www.olympedia.org/): PyeongChang 2018, Tokyo
2020, Beijing 2022, Paris 2024, and Milano-Cortina 2026.

## Datasets

- [`olympic_athletes`](https://moderndive.github.io/olympicAthletes/reference/olympic_athletes.md):
  ~315,000 athlete-event rows.

- [`medal_table`](https://moderndive.github.io/olympicAthletes/reference/medal_table.md):
  verified medal tables for every edition, 1896-2026.

- [`editions`](https://moderndive.github.io/olympicAthletes/reference/editions.md):
  edition-level metadata for every Games, 1896-2026.
